1. What's the difference between map, forEach, filter and find?

1. map(): map is array method. Looping over each element of an array and does something then it returns results inside a new array. If need results later then we will use map.

2. forEach(): forEach is array method. Looping over each element of an array and does something directly. forEach doesn't return any valid results. forEach returns undefined. If don't need results later then we will use forEach.

3. filter(): filter is array method. Looping over each element of an array and does check with condition. If condition is meets then returns all element inside a new array the elements that fulfilled the condition.

4. find(): find is array method. Looping over each element of an array and does check with condition. If condition is meets then returns only first element the elements that fulfilled the condition.

2. What's the difference between localStorage, sessionStorage, and cookies?

1. localStorage: localStorage is browser api. It is used for storing data that the user will need to access later. If the website, browser or pc is closed or reloaded, the data remains inside localStorage. localStorage has no expiration date.

2. sessionStorage: sessionStorage is browser api. It is used for storing data that the user will need to access up to a certain time. If the website or browser reloaded, the data remains inside sessionStorage. But website, browser or pc is closed the data is not remains. sessionStorage has expiration date.

3. cookies: cookies is browser api. It is used for storing data, such as: login information, users activity etc up to a certain time. There are two types of cookies, first party cookie and third party cookie. cookie has expiration date.
